# 🛒 Shopping Cart Application

A modern and responsive Shopping Cart web application built using **React.js**.  
This project allows users to browse products, add items to cart,  and view total price dynamically.

## 🚀 Features

- 🛍️ Product Listing
- ➕ Add to Cart
- ➖ Remove from Cart
- 💰 Dynamic Total Price Calculation
- ⚡ Fast and Responsive UI
- 📱 Mobile Friendly Design
- 🔄 Real-time Cart Updates
